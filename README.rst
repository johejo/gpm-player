gpm-player
==========

Description
-----------

Multi platform command-line-player of Google Play Music using gmusicapi.

Environment
-----------

Python3 on Windows 10 and POSIX.

Requirements
------------

- VLC (libvlc)

Install
-------

.. code:: bash

    $ pip install git+https://github.com/johejo/gpm-player

Usage
-----

.. code:: bash

    $ gpm-player

Future help

.. code:: bash

    $ gpm-station --help
