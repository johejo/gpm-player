Description
-----------

Multi platform command-line-player of Google Play Music using gmusicapi.

Environment
-----------

Python3 on Windows 10 and POSIX.

Requirements
------------

- VLC (libvlc)

Install
-------

.. code:: bash

    $ pip install gpm-player

Usage
-----

station player
~~~~~~~~~~~~~~

.. code:: bash

    $ gpm-station

playlist player
~~~~~~~~~~~~~~~

.. code:: bash

    $ gpm-playlist

Tasks
-----

- equalizer
- seek and set time
